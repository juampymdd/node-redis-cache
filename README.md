# Node + Redis (cache)

This is a simple example to show how to use Redis as a cache in a Node.js application.

1. Clone repository

2. Install dependencies

```bash
npm install
```

3. init docker-compose to start Redis

```bash
docker-copose up -d
```

4. run the application

```bash
# Normal mode
node index.js

# Watch mode
node --watch index.js
```
